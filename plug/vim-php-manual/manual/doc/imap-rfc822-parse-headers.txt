*Description*

==============================================================================
*Parameters*

{headers}

    The parsed headers data

{defaulthost}

    The default host name

==============================================================================
*Return* *Values*

Returns an object similar to the one returned by |imap_header|, except for the
flags and other properties that come from the IMAP server.
