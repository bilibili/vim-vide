(PECL ps >= 1.3.0)

ps_shfill â€” Fills an area with a shading

==============================================================================
*Description*

    bool ps_shfill ( resource $psdoc , int $shadingid )~

Fills an area with a shading, which has to be created before with |ps_shading|
. This is an alternative way to creating a pattern from a shading 
|ps_shading_pattern| and using the pattern as the filling color.

==============================================================================
*Parameters*

{psdoc}

    Resource identifier of the postscript file as returned by |ps_new|.

{shadingid}

    The identifier of a shading previously created with |ps_shading|.

==============================================================================
*Return* *Values*

Returns TRUE on success or FALSE on failure.

==============================================================================
*See* *Also*

  * |ps_shading| - Creates a shading for later use
  * |ps_shading_pattern| - Creates a pattern based on a shading
