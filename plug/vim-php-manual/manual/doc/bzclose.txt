==============================================================================
*Description*

    int bzclose ( resource $bz )~

Closes the given bzip2 file pointer.

==============================================================================
*Parameters*

{bz}

    The file pointer. It must be valid and must point to a file successfully
    opened by |bzopen|.

==============================================================================
*Return* *Values*

Returns TRUE on success or FALSE on failure.

*See* *Also*
