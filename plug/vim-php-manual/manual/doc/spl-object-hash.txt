spl_object_hash

(PHP 5 >= 5.2.0, PHP 7)

==============================================================================
*Description*

    string spl_object_hash ( object $obj )~

This function returns a unique identifier for the object. This id can be used
as a hash key for storing objects, or for identifying an object, as long as
the object is not destroyed. Once the object is destroyed, its hash may be
reused for other objects.

==============================================================================
*Parameters*

{object}

    Any object.

==============================================================================
*Return* *Values*

A string that is unique for each currently existing object and is always the
same for each object.

==============================================================================
*Examples*

Example #1 A spl_object_hash() example

<?php >
    $id = spl_object_hash($object);
    $storage[$id] = $object;
<?>

==============================================================================
*Notes*

    Note:

    When an object is destroyed, its hash may be reused for other objects.
