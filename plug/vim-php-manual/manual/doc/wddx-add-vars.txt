==============================================================================
*Parameters*

    A WDDX packet, returned by |wddx_packet_start|.

{var_name}

    Can be either a string naming a variable or an array containing strings
    naming the variables or another array, etc.

{...}

==============================================================================
*Return* *Values*

Returns TRUE on success or FALSE on failure.
