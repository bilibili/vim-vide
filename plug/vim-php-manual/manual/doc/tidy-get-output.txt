tidy_get_output

(PHP 5, PHP 7, PECL tidy >= 0.5.2)

tidy_get_output â€” Return a string representing the parsed tidy markup

==============================================================================
*Description*

    string tidy_get_output ( tidy $object )~

Gets a string with the repaired html.

==============================================================================
*Parameters*

{object}

    The Tidy object.

==============================================================================
*Return* *Values*

Returns the parsed tidy markup.

==============================================================================
*Examples*

Example #1 tidy_get_output() example

<?php >

    $html = '<p>paragraph</i>';
    $tidy = tidy_parse_string($html);

    $tidy->cleanRepair();

    echo tidy_get_output($tidy);
<?>

The above example will output:

<!DOCTYPE html PUBLIC "-//W3C//DTD HTML 3.2//EN">
<html>
<head>
<title></title>
</head>
<body>
<p>paragraph</p>
</body>
</html>
