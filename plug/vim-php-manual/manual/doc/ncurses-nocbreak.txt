==============================================================================
*Description*

    bool ncurses_nocbreak ( void )~
Warning

This function is EXPERIMENTAL. The behaviour of this function, its name, and
surrounding documentation may change without notice in a future release of

Returns terminal to normal (cooked) mode. Initially the terminal may or may
not be in cbreak mode as the mode is inherited. Therefore a program should
call |ncurses_cbreak| and ncurses_nocbreak() explicitly.

==============================================================================
*Return* *Values*

Returns TRUE if any error occurred, otherwise FALSE.
