*Description*

==============================================================================
*Parameters*

{connection_id}

    The ODBC connection identifier, see |odbc_connect| for details.

    The SQL statement.

{flags}

    This parameter is currently not used.

==============================================================================
*Return* *Values*

Returns an ODBC result identifier if the SQL command was executed
successfully, or FALSE on error.
