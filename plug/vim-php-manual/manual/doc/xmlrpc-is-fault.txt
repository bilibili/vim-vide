*Parameters*

{arg}

    Array returned by |xmlrpc_decode|.

==============================================================================
*Return* *Values*

Returns TRUE if the argument means fault, FALSE otherwise. Fault description
is available in $arg["faultString"], fault code is in $arg["faultCode"].
