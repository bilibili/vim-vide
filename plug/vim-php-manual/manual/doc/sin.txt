==============================================================================
*Description*

    float sin ( float $arg )~

sin() returns the sine of the {arg} parameter. The {arg} parameter is in
radians.

==============================================================================
*Parameters*

{arg}

    A value in radians

==============================================================================
*Return* *Values*

The sine of {arg}

==============================================================================
*Examples*

Example #1 sin() example

<?php >

    // Precision depends on your precision directive
    echo sin(deg2rad(60));  //  0.866025403 ...
    echo sin(60);           // -0.304810621 ...

<?>
