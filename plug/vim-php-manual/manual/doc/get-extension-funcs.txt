get_extension_funcs

(PHP 4, PHP 5, PHP 7)

get_extension_funcs â€” Returns an array with the names of the functions of a
module

==============================================================================
*Description*

    array get_extension_funcs ( string $module_name )~

This function returns the names of all the functions defined in the module
indicated by {module_name}.

==============================================================================
*Parameters*

{module_name}

    The module name.

        Note:

        This parameter must be in lowercase.

==============================================================================
*Return* *Values*

Returns an array with all the functions, or FALSE if {module_name} is not a
valid extension.

==============================================================================
*Examples*

Example #1 Prints the XML functions

<?php >
    print_r(get_extension_funcs("xml"));
<?>
